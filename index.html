HTML: Defining the Webpage Structure

  • The Head element contains website metadata.
  • The Body element represents visible content rendered by the user browser.

CSS: Defining the Webpage Style

  • Margin defines the whitespace surrounding an Element.
  • Padding defines the whitespace surrounding Content within an Element.

Git: Storing and Synchronizing Project Code

  • Git Repository Management Progressions:
    1. cd /'project-directory'
    2. git status
    3. git checkout main
    4. git pull origin main
    5. git checkout -b feature/'new-branch-name'
    6. git status
    7. git add -A
    8. git commit -am "Description of Code Updates"
    9. git pull origin main
    10. git push origin 'feature/new-branch-name'
    11. GitHub: Process Pull Request
    12. Miscellaneous Note: Insert Git Commit Note - Type commit message, press escape, type :wq , which will *w*rite the file (save it) and *q*uit Vi/Vim.
